== Description ==

Override your titles, descriptions and keywords with page/post level input for greater control of **SEO**.

*Rush SEO Features*

* Modify meta deta for each page or post.
* Auto-generate your titles, descriptions and keywords when overrides are not present. ( Optional )
* Helps prevent duplication content issues.
* Automatically appends tags as keywords when available.
* Makes sure your keyword(s) are unique.

You can learn more about Rush SEO on our **[website](http://wprush.blog.co.uk/)** website.

== Installation ==

1. Unzip into your `/wp-content/plugins/` directory and make sure the files are inside the rush-seo/ folder.
1. Activate the plugin through the Wordpress Panel -> 'Plugins'
1. Visit (*Settings - Rush SEO*)
1. That's it.
